extract-text-webpack-plugin

该插件的主要是为了抽离 css 样式,防止将样式打包在 js 中引起页面样式加载错乱的现象。
生产环境下比较推荐的做法是，使用 ExtractTextPlugin 将样式表抽离成专门的单独文件。这样，样式表将不再依赖于 JavaScrip

ExtractTextPlugin.extract

ExtractTextPlugin.extract 从一个已存在的 loader 中，创建一个提取(extract) loader。

  • options.use 该 loader 将资源转换成一个 CSS 导出模块;
  • options.fallback 应用于当 CSS 没有被提取时，此 loader 将资源转换成一个 CSS 导出模块；
  • options.publicPath 重写此 loader 的 publicPath 配置
