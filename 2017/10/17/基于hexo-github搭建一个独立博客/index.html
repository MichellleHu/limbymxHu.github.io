将 ssh key 添加到 Github 中

Find 前往文件夹~/.ssh/id_rsa.pub 打开 id_rsa.pub 文件，里面的信息即为 SSH key，将这些信息复制到 Github 的 Add SSH key 页面即可。

进入 Github –> Settings –> SSH keys –> add SSH key:

Title 里任意添一个标题，将复制的内容粘贴到 Key 里，点击下方 Add key 绿色按钮即可。
