call和apply

javascript 中通过 call 或者 apply 用来代替另一个对象调用一个方法，将一个函数的对象上下文从初始的上下文改变为由 thisObj 指定的新对象。简单的说就是改变函数执行的上下文，这是最基本的用法。两个方法基本区别在于传参不同。

call(obj,arg1,arg2,arg3); call 第一个参数传对象，可以是 null。参数以逗号分开进行传值，参数可以是任何类型。
apply(obj,[arg1,arg2,arg3]); apply 第一个参数传对象，参数可以是数组或者 arguments 对象。
